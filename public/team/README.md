## Notes
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
  (if you use a different extension, update the `photo` field in `MeetTheTeam.tsx`)
- Recommended size: **400×400 px** minimum, square crop
- Until a file is placed here, the card automatically falls back to the placeholder silhouette — no broken-image icons will appear
